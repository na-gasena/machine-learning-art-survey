# データセット・ツール・資料

## データセット

- ImageNet: 視覚認識研究の基盤。批評的AIアートでも重要。https://www.image-net.org/
- WikiArt: 美術画像・様式分類・GAN/CAN研究で多用される。https://www.wikiart.org/
- LAION-5B: 大規模画像テキストデータセット。生成AIの訓練データ倫理の中心事例。https://arxiv.org/abs/2210.08402
- NSynth: 注釈付き音楽単音データセット。https://magenta.tensorflow.org/datasets/nsynth
- ArtEmis: 美術作品と感情・言語説明を結びつけるデータセット。https://www.artemisdataset.org/
- People-Art / Human-Art: 絵画・人工シーンにおける人物検出や姿勢推定に関わるデータセット。https://arxiv.org/abs/1610.08871 / https://arxiv.org/abs/2303.02760
- ArtBench-10: 美術画像生成モデルのベンチマーク。https://github.com/liaopeiyuan/artbench
- BAM! Behance Artistic Media Dataset: 写真以外の芸術的メディア認識データセット。https://arxiv.org/abs/1704.08614
- FFHQ: 顔生成・GAN史で重要な顔画像データセット。The Zizi Projectのようなデータセット介入を考える際にも参照される。https://github.com/NVlabs/ffhq-dataset

## ツール

- Wekinator: アーティスト・音楽家向けインタラクティブ機械学習。https://doc.gold.ac.uk/~mas01rf/Wekinator/
- Magenta: 音楽・アート向け創作機械学習プロジェクト。https://magenta.tensorflow.org/
- Stable Diffusion: オープンウェイト系テキスト画像生成エコシステム。https://github.com/CompVis/stable-diffusion
- Glaze: 作家のスタイル模倣対策ツール。https://glaze.cs.uchicago.edu/
- C2PA tools/specifications: 生成・編集メディアの来歴標準。https://c2pa.org/
- ml5.js: ブラウザ上のクリエイティブ機械学習ライブラリ。https://ml5js.org/
- Teachable Machine: コードなしで分類モデルを訓練できる教育/プロトタイピング環境。https://teachablemachine.withgoogle.com/
- ml4a: アーティスト向け機械学習教育リソース。https://ml4a.github.io/
- Runway: 生成動画・映像制作向けAIプラットフォーム。https://runwayml.com/

## 利用上の注意

データセットやモデルを再利用する場合は、ライセンス、対象画像・音声の権利、人物データの同意、地域法、公開時の表示義務を個別に確認してください。とくにLAION-5B、WikiArt派生データ、人物画像を含むデータセットは、研究利用であっても倫理・権利上の検討が必要です。
